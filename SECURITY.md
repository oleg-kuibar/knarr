# Security Policy

## Supported Versions

Knarr follows the latest published version on npm. Security fixes are released as patch versions when possible.

## Reporting A Vulnerability

Please do not open a public issue for security reports.

Email Oleg Kuibar at <oleg.kuibar@proton.me> with:

- A short description of the issue
- Steps to reproduce or a proof of concept
- Affected versions, if known
- Any suggested mitigation

You should receive an initial response within 7 days.
